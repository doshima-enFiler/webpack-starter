# webpack-starter

Webpackとnpm scriptsを使用した、Flocss × Dart Sassのコーディング環境。

## How to use

1. リポジトリをクローン
`git clone https://github.com/oshima-daiki/webpack-starter`

2. 開発用のパッケージをインストール
`npm i`

3. コンパイルやローカルサーバーの立ち上げなどを実行
`npm run w`

コンパイルだけなら`npm run build`で可能。
